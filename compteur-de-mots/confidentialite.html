Politique de Confidentialité

Dernière mise à jour : 20 avril 2026

1. Introduction

Cette politique de confidentialité décrit comment l'extension Chrome « Compteur de Mots et Caractères » traite vos données. Nous nous engageons à protéger votre vie privée conformément au Règlement Général sur la Protection des Données (RGPD) et aux recommandations de la CNIL.

2. Responsable du traitement

Le responsable du traitement des données est l'éditeur de l'extension « Compteur de Mots ». Pour toute question, vous pouvez nous contacter via la page de support de l'extension sur le Chrome Web Store.

3. Données collectées

Nous ne collectons aucune donnée personnelle.

L'extension utilise exclusivement :

  • Stockage local (chrome.storage.local) — pour sauvegarder temporairement le texte sélectionné via le menu contextuel, l'état du bandeau d'évaluation et un identifiant anonyme Google Analytics. Ces données restent uniquement sur votre appareil.

Le texte que vous analysez n'est jamais envoyé sur un serveur. Tout le comptage (mots, caractères, phrases, paragraphes, densité des mots-clés) est effectué localement dans votre navigateur.

L'extension ne collecte pas :

  • Aucune donnée d'identification personnelle
  • Aucune donnée de navigation ou d'historique web
  • Aucun contenu de page web (sauf le texte que vous sélectionnez explicitement pour le compter)
  • Aucun cookie de tracking
  • Aucune adresse IP
  • Aucune donnée de géolocalisation

4. Permissions Chrome

L'extension demande uniquement les permissions minimales nécessaires :

  • contextMenus — pour ajouter l'option « Compter les mots » dans le menu contextuel Chrome.
  • storage — pour sauvegarder temporairement le texte sélectionné, les préférences et l'identifiant anonyme GA4.

L'extension ne demande pas de permissions sensibles telles que tabs, history, bookmarks, webNavigation, ou l'accès au contenu de vos pages web. Le texte n'est lu que lorsque vous le sélectionnez explicitement et utilisez le menu contextuel.

5. Analyse d'utilisation (Google Analytics 4)

L'extension envoie des données d'utilisation anonymes à Google Analytics 4 afin d'améliorer ses fonctionnalités :

  • Installation et mises à jour de l'extension
  • Utilisation de la fonction « compter les mots » (sans contenu du texte)

Ne sont pas collectés : le contenu de votre texte, vos données personnelles, votre historique de navigation, votre adresse IP ou toute autre information permettant l'identification. Toutes les données analytiques sont entièrement anonymisées.

6. Transfert de données

Seules des statistiques d'usage anonymes sont transmises à Google Analytics. Aucune donnée personnelle n'est transférée vers des tiers.

7. Durée de conservation

Les préférences locales sont conservées jusqu'à ce que vous désinstalliez l'extension. Le texte sélectionné via le menu contextuel est supprimé automatiquement dès que vous l'analysez.

8. Vos droits (RGPD)

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès — toutes vos données locales sont visibles directement dans l'extension.
  • Droit de suppression — désinstallez l'extension pour supprimer toutes les données locales.
  • Droit à la portabilité — non applicable (aucune donnée personnelle collectée).
  • Droit d'opposition — vous pouvez désactiver Google Analytics dans les paramètres de votre navigateur.

9. Mineurs

L'extension ne s'adresse pas spécifiquement aux mineurs et ne collecte aucune donnée permettant d'identifier l'âge de l'utilisateur.

10. Modifications

Cette politique peut être mise à jour. La date de dernière modification figure en haut de cette page.

11. Contact

Pour toute question relative à cette politique de confidentialité, vous pouvez nous contacter via la page de support de l'extension sur le Chrome Web Store.
